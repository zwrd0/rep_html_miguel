HTML, O QUE É e PARA QUE SERVE?

HTML é uma abreviação para "HyperText Markup Language". Documentos HTML servem para a criação de páginas web.

COMO CRIAR UMA PÁGINA HTML?

Substituindo todos os "[]" por "<>"

✽ 1º Passo: Criar um ficheiro do tipo "nome_do_ficheiro.html".

✽ 2º Passo: Abrir o ficheiro num editor de texto, como por exemplo, "Visual Studio Code" ou "Notepad++".

✽ 3º Passo: Escrever na primeira linha "[!DOCTYPE html]".

✽ 4º Passo: Abrir um bloco com "[html] [/html]".

✽ 5º Passo: Dentro desse bloco criar um outro com "[body] [/body]", este bloco determina que tudo dentro faz parte do corpo do texto HTML.

✽ 6º Passo: Dentro desse mesmo bloco "body", criar um novo bloco com [h1] e [/h1] se quiser criar um novo título, ou um com [p] e [/p] para criar um novo parágrafo!

Agora é so abrir o ficheiro e deve abrir uma página no navegador com o texto!
